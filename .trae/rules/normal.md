# 项目介绍
项目开源地址：https://github.com/yahao333/terms-and-conditions
功能是提供paddle网站需要的Terms of service，该服务的说明：
URL must include a path, e.g. website.com/terms-and-conditions

当前项目是提交到 github仓库后通过 vercel 部署到 https://terms-and-conditions.vercel.app/

当前web网站是单页面的网站.

## 技术栈

- react
- typescript
- vercel
- vite
